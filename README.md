# Sentinel Analytics Rules JSON Editor

## Overview
This project provides a simple web-based JSON editor designed to edit and validate **Microsoft Sentinel Analytics Rules** exported in JSON format. The editor is built using **HTML**, **JavaScript**, and **Tailwind CSS** via CDN for a clean and responsive UI.

## Features
- Load JSON files exported from Microsoft Sentinel.
- Edit JSON content directly in a text area.
- Validate JSON structure before saving.
- Download the updated JSON file.
- Responsive design powered by Tailwind CSS.

## How to Use
1. Clone or download this repository.
2. Open the `index.html` file in your browser.
3. Use the **Load Folder** button to load the JSON files.
4. Edit the JSON in the provided text area.
5. Click **Save JSON** to validate and store changes.

## Requirements
- A modern web browser (Chrome, Edge, Firefox).
- No server required; runs entirely in the browser.
- Internet connection to load Tailwind CSS via CDN.

## Technologies Used
- **HTML5** for structure.
- **Tailwind CSS** (via CDN) for styling.
- **Vanilla JavaScript** for functionality.
